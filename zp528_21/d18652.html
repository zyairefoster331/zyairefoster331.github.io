Nvidia Jetson Nano / Xavier NX / Raspberry Pi用 12.3MPカメラボード

概要

  • Nvidia Jetson Nano / Xavier NX / Raspberry Pi用 12.3MPカメラボード
  • より良い画質とレンズの柔軟性を提供します。
  • ハードウェアの変更は必要ありません。
  • NVIDIA Argusカメラプラグインに対応しています。
  • IR感度: 可視光
  • インタフェース: 4レーン MIPI CSI-2

Nvidia Jetson Nano / Xavier NX / Raspberry Pi用 12.3MPカメラボードは、3穴ピッチの IMX477カメラボードです。

このカメラモジュールは、ISP機能を備えた Jetson Nano / Xavier NXでネイティブに使用できます。ハードウェアを変更する必要はありません。ドライバをインストールするだけで使えます。

IMX219の見逃せないアップデートを備えています。- 53％多いピクセル 92％大きいピクセルサイズ 192％大きいイメージング領域 CSマウントで、はるかに優れた画質とレンズの柔軟性が得られます。最大静止解像度は 4056 x 3040 フレームレートは 30fps@Full 12.3MP 60fps@1080Pです。

注: C/CSレンズは含まれていません。Pi 2 / 3 / 4 / Zeroでは動作しません。コンピュートモジュールだけで動作します。カメラドライバをインストールする必要があります。標準のRaspberry Piモデル (Raspberry Pi 3 / 3B+ / 4など) では機能しません。

仕様

  • センサ: IMX477
  • 光学フォーマット: 1/2.3インチ (対角 7.857mm)
  • 解像度: 4056 x 3040 (横 x 縦) 12.3MP
  • IR感度: 可視光
  • インタフェース: 4レーン MIPI CSI-2
  • デフォルトのレンズマウント: CSマウント
  • フレームレート:
    • 60fpsで 1920 x 1080
    • 30fpsで 4032 x 3040
