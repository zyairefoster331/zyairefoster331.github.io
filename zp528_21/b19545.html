【USB C KVM切替器】この2ポートUSB C切替器により、2つのUSB Cビデオソースが1つのHDMIディスプレイを共有できます。☆注意: この USB C KVM スイッチ セレクターを購入する前に、お使いのコンピューター/デバイスに少なくとも 1 つの Type C ポートがあることを確認してください。 (データ、ビデオ、PD をサポート、つまり、Displayport Alt モードがサポートされています)
【Ultra HD 4K@60Hzの視覚的な楽しみ】この USB C to HDMI KVM スイッチ デバイスは最大4K@60Hzの解像度をサポートし。 出力の最大解像度とリフレッシュ レートは、ソース デバイスとケーブルによって異なります。 4K 解像度が必要な場合は、ソースデバイス、モニター、ケーブルが 4K 対応であることを確認してください。
【2つの切り替えモード】このKVM 切替器はドライバー不要、プラグアンドプレイです。 2つの切り替え方法が用意されています。 パネル上のボタンを押すか、有線リモコン (パッケージに含まれています) を使用して、コンピューターを切り替えることができます。 LED はアクティブなコンピュータを示します。
【優れた互換性】このUSB-C to HDMI切替器は、Windows 10/8/8.1/7/XPおよびMac OS、Linuxシステムと互換性があります。 ワイヤレス キーボード、マウス、を認識する USB 2.0 ポート。 さらに多くの USB デバイスを使用したい場合は、USB ハブを介して複数の USB ポートを拡張できます。
【商品内容】 AOOCOO チームは、この USB C KVM スイッチに 1 年間の保証と 24 時間の電子メール サポートを提供します。パッケージ内容: 1x USB C KVM 切替器、2x USB C ケーブル、1x 有線リモコン、1x DC5V 電源ケーブル、1x 取扱説明書 。
