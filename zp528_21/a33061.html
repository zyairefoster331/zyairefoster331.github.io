生産者：ペレ=ヴェルジェ社は、1985年、カトリーヌ・ペレ＝ヴェルジェによってポムロールに設立された家族経営の企業です。巨大資本の傘下に入るグランシャトーが多くなっている昨今、このレベルのシャトーを家族規模で経営している最後の企業といえます。設立当初からカトリーヌと親交の深かった醸造コンサルタントのミッシェル・ロランが全面的にサポートし、世界に誇れる品質のワイン造りを行ってきました。現在は、子息のアンリ・パランが亡きカトリーヌの志を受け継ぎ、一層の情熱を持ってワイナリーを運営しています。自然環境保護への取り組みも重視し、2020年にHVE（環境価値重視）認証を取得しました。 シャトー・ル・ゲのセカンドワイン。ブラックチェリー、プラム、スミレ、杉、スパイスなど複雑で深い香り。力強いタンニンとエレガントさが共存した味わい。 畑：ポムロール最上のテロワールをもつ高台に4ha。南東向き。下層にこの地方特有の青錆色の斑点がある粘土、表面には砂礫を含んだ粘土質土壌。 栽培・収穫：樹齢15年。植密度9,900本/ha、収量39hl/ha。収穫は天候により9月中旬 10月初め。 醸造・熟成：発酵前に10℃で7日間の果皮浸漬。円錐型コンクリートタンクにて25日間の発酵。天然酵母使用。マロラクティック発酵と熟成は一年樽で12カ月。醸造コンサルタントはミッシェル・ロランとマルチェロ・ペレリッティ。年産20,000本。 受賞歴：2009 ワインアドヴォケイト 2012 90pt Web 2010 ワインスペクテーター 91pt ( 2013/03 ) 2011 ワインスペクテーター 2012 89 92pt Web 2012 ワインスペクテーター 2013 90 93pt Web
